## 담당업무

1. 회원(User)관리 담당
2. 회원가입, 로그인, 회원정보 수정 페이지 담당
3. 발표 담당

## 기술

 
 - Java 

- JavaScript

- Oracle SQL

- MyBatis

- JSP

## 프로젝트 소개

- 현재 운영되고 있는  쇼핑몰 사이트인 kream 사이트를 참고

- 회원, 상품, 카트, 주문 4개의 파트로 나눠서 프로젝트 진행 

- 상품을 브랜드별로 나눠서 페이지 목록 구성

- 상품에서 직접 주문하기와 상품을 카트에 담은 후 주문하기 기능 둘다 구현
